# timeline
时间轴组件(Angular)
垂直展示的时间流信息。

# 何时使用
当有一系列信息需按时间排列时，可正序和倒序。
需要有一条时间轴进行视觉上的串联时。

# 代码摘自NG-ZORRO组件
